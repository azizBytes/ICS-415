# 🧱 Project 3: Voxel Engine

## 📌 Description

This project simulates a **Minecraft-style voxel world**, allowing you to navigate and manipulate 3D blocks using Java and OpenGL.

## 🎯 Features

- Chunk-based terrain generation
- Player movement (WASD + mouse look)
- Block placement/removal with click
- Lighting and shadows
- Texture-mapped blocks

## 📂 Folder Contents

- `engine/`: Core rendering logic
- `terrain/`: Chunk and block data
- `input/`: Keyboard and mouse handling
- `Main.java`: Entry point

## 🛠 How to Run

1. Install LWJGL and link to your IDE.
2. Compile and run `Main.java`.

## 🕹 Controls

- `WASD`: Move
- `Mouse`: Look around
- `Left Click`: Remove block
- `Right Click`: Place block

## 🏞 Example

> Fly through a voxel world with dynamic terrain and build your own structures.
